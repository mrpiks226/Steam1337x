# 🎮 Steam1337x
Adds a custom **🔍 Search 1337x** button directly to game pages on **Steam** and **Epic Games Store**.

---

## 📥 Installation

Choose your browser below to see the instructions:

<details>
<summary><b>🌐 Google Chrome / Chromium Browsers</b></summary>
<br>

1. **Download** the repository as a ZIP and extract it.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer Mode** (top right toggle).
4. Click **Load unpacked** and select the folder containing the files.
</details>

<details>
<summary><b>🦊 Mozilla Firefox (Permanent Install)</b></summary>
<br>

1. **Download the signed file:** [Click here to download steam1337x.xpi](./steam1337x.xpi)
2. Open Firefox and type `about:addons` in the address bar.
3. Click the **Gear Icon** (⚙️) in the top-right menu.
4. Select **"Install Add-on From File..."**
5. Select the `.xpi` file you just downloaded.
6. Click **Add** when the popup appears.
</details>

---

## 🛠 Features
* **Steam Support:** Adds button next to the game title.
* **Epic Games Support:** Automatically handles "soft-loading" (button stays visible even when switching games).
* **Smart Search:** Searches specifically within the **Games** category on 1337x.

## 📜 Technical Details
* **Manifest:** V3 (Cross-browser compatible)
* **Permissions:** `activeTab` (Only runs when you are looking at the page)
